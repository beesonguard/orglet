# Contributing to Orglet

Thanks for your interest in contributing to Orglet!

## Project Status

Orglet is in early development. The foundation is still being built, so large contributions may be premature. However, feedback and ideas are welcome.

## Ways to Contribute

### Right Now
- **Feedback**: Open an issue with suggestions or feature requests
- **Bug reports**: If you're testing early builds, report issues
- **Documentation**: Help improve docs, fix typos

### Once MVP is Further Along
- **Code contributions**: Bug fixes, features, improvements
- **Testing**: Help test on different Android devices
- **Translations**: Help localize the app

## Code Style

- TypeScript with strict mode
- Functional components with hooks
- Prettier for formatting (config coming)

## Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description
